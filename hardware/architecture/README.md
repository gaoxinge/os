## 芯片

### 工艺

- x86: CISC
- arm: RISC

### 分类

- CPU
- GPU
- TPU
- FPGA
- ASIC

### 厂商

- Intel
- AMD

### 框架

- OpenCL: 提供跨平台的支持
- CUDA: 只支持NVIDIA的GPU

## 显卡

### 分类

- N卡: NVIDIA, 和Intel合作
- A卡: ATI, 被AMD收购
- Intel(R) HD Graphics

### 框架

- OpenGL: 跨平台图形库
- DirectX: Windows上的多媒体库
- 游戏引擎: 架构在OpenGL或者DirectX上
- rpgmaker: ruby, javascript
- tkinter: pygame
